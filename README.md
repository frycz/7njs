# 7njs

### Setup

```
npm install
npm start
start browser with http://localhost:8080
```

### Other tasks


build dev bundle

```
npm run build
```

build bundle for production

```
npm run buildProduction
```

run server

```
npm run start
```

run production server

```
npm run startProduction
```

lint code

```
npm run lint
```
